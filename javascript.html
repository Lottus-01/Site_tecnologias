JAVASCRIPT

JavaScript é uma linguagem de programação usada principalmente para adicionar interatividade a páginas web, tornando-as dinâmicas e responsivas. É uma das tecnologias fundamentais da web, permitindo funcionalidades como animações, menus interativos, atualizações em tempo real e muito mais. Além do uso em navegadores (client-side), JavaScript também pode ser usado no lado do servidor (back-end) com Node.js, e em outras plataformas
