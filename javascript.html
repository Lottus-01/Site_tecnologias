JAVASCRIPT

JavaScript é uma linguagem de programação usada principalmente para adicionar interatividade a páginas web, tornando-as dinâmicas e responsivas. É uma das tecnologias fundamentais da web, permitindo funcionalidades como animações, menus interativos, atualizações em tempo real e muito mais. Além do uso em navegadores (client-side), JavaScript também pode ser usado no lado do servidor (back-end) com Node.js, e em outras plataformas

Para que serve o JavaScript?
Interatividade:
Permite que as páginas web reajam a ações do usuário, como cliques, envios de formulários e movimentação do mouse.
Conteúdo dinâmico:
Atualiza partes da página sem a necessidade de recarregar toda a página, como mostrar notificações ou resultados de busca em tempo real.
Manipulação de elementos HTML e CSS:
Permite modificar a estrutura e o estilo da página dinamicamente.
Criação de aplicativos web:
Frameworks como React, Angular e Vue usam JavaScript para construir interfaces de usuário complexas e interativas.
Desenvolvimento back-end:
Com Node.js, JavaScript pode ser usado para criar servidores e aplicativos web.
