CSS

CSS, ou Cascading Style Sheets, é uma linguagem de estilo usada para descrever a apresentação de documentos escritos em HTML ou XML (como páginas web). Em termos simples, CSS controla como os elementos de uma página web são exibidos, como cores, fontes, espaçamento, layout e muito mais.

O que o CSS faz? Estilização: Permite alterar a aparência de elementos HTML, como cores, fontes, tamanhos, margens e fundos. Layout:
Define como os elementos são dispostos na página, incluindo a organização em colunas, linhas e a posição de cada elemento. Design responsivo:
Permite criar layouts que se adaptam a diferentes tamanhos de tela (desktops, tablets, celulares).
Separação de conteúdo e apresentação
Mantém o conteúdo HTML separado do estilo, facilitando a manutenção e a atualização do site.
