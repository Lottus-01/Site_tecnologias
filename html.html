HTML

HTML (HyperText Markup Language) é uma linguagem de marcação usada para estruturar o conteúdo de páginas web. Não é uma linguagem de programação, mas sim um conjunto de elementos que indicam como o navegador deve exibir o conteúdo. Esses elementos, como <p> para parágrafos ou <img> para imagens, são usados para organizar e formatar textos, imagens, links e outros tipos de conteúdo em uma página

Em termos mais simples, o HTML funciona como o esqueleto de um site. Ele define a estrutura, como parágrafos, listas, títulos e outros elementos que compõem a página. O HTML também permite a inserção de imagens, vídeos e outros tipos de mídia, além de links para outras páginas ou recursos.
