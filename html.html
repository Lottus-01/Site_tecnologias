HTML

HTML (HyperText Markup Language) é uma linguagem de marcação usada para estruturar o conteúdo de páginas web. Não é uma linguagem de programação, mas sim um conjunto de elementos que indicam como o navegador deve exibir o conteúdo. Esses elementos, como
para parágrafos ou para imagens, são usados para organizar e formatar textos, imagens, links e outros tipos de conteúdo em uma página
